# Forma Site Design SDK Skills

AI agent skills for building [Autodesk Forma](https://www.autodesk.com/products/forma) site design extensions.

## Installation

Skills can be installed as personal skills (available across all projects) or project skills (specific to one repository).

### GitHub Copilot CLI

Copy the skills to your personal skills directory:

```bash
cp -r skills/* ~/.copilot/skills/
```

Or for project-level installation:

```bash
cp -r skills/* .github/skills/
```

See [About Agent Skills](https://docs.github.com/en/copilot/concepts/agents/about-agent-skills) for more details.

### Claude Code CLI

Copy the skills to your personal skills directory:

```bash
cp -r skills/* ~/.claude/skills/
```

Or for project-level installation:

```bash
cp -r skills/* .claude/skills/
```

See [Claude Code Skills](https://code.claude.com/docs/en/skills) for more details.
